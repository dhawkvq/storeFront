# StoreFront

## Prerequisites

<div style='border:1px solid red; width: 80%; margin: 10px auto;'></div>

**Node version 16.x**

```
  If you use volta for node management you are set to go.
```

## Run Project

<div style='border:1px solid green; width: 80%; margin: 10px auto;'></div>

**Install dependencies**

```
yarn
```

**Start dev server**

```
yarn start
```
